# Flutter E-commerce Bootcamp

## Quick Summary
Certificate of completion for passing the **Flutter E-commerce Bootcamp** organized by **Android Enthusiast Jakarta**. This certificate reflects structured hands-on learning in building Flutter applications around e-commerce use cases.

## Issuer
- Android Enthusiast Jakarta
- Signed by **Fiqri Hafzain Islami**, Co-Founder of Android Enthusiast Jakarta

## What This Certificate Represents
- Completed a bootcamp focused on Flutter-based application development.
- Built understanding of mobile app flow in an e-commerce context.
- Learned implementation approaches such as clean architecture and modularization using `melos`.
- Strengthened practical foundation in UI implementation, app structure, and feature delivery with Flutter.
- Practiced development best practices for organizing code, separating responsibilities, and keeping projects easier to scale.

This certificate marks one of the earlier milestones in the journey of learning Flutter seriously. It shows commitment to learning not only feature building, but also maintainable architecture patterns, modular project organization, and better engineering best practices for long-term app development.
